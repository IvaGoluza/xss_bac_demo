# XSS-BAC Demo App

## Project Overview
This is a web application that allows a potential attacker to use two different security attack techniques; web application vulnerabilities.<br/>
<br/>
**Implemented  vulnerabilities:**
- Cross-site scripting (XSS)
- Broken Access Control<br/>
For each, there is a functionality that enables/disables vulnerability.

## Technology Stack and Deployment 
This is a [Next.js](https://nextjs.org/) project bootstrapped with [`create-next-app`](https://github.com/vercel/next.js/tree/canary/packages/create-next-app).<br/>
The application is deployed on Vercel. The deployed application can be accessed at https://xss-bac-demo.vercel.app/

## Setup Instructions
First, run the development server:
```bash
npm run dev
```
Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
